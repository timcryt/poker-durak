Как играть

  1. Зайдите на страницу игры
  2. Зайдите на неё ещё раз, если всё пройдёт успешно, то вам будет выдан список карт, и вы сможете начать играть
  3. Если вам пришло сообщение YourTurn, значит ход ваш

  4. Есть 4 варианта хода

    1. {"MakeStep": "GetCard"} - взять карту (доступен только при State Passive)
    2. {"MakeStep": {"GiveComb": {карты}}} - выложить комбинацию (доступен только при State Passive)
    3. {"MakeStep": {"TransComb": {карты}}} - перевести комбинацию (доступен только при State Active)
    4. {"MakeStep": "GetComb"} - взять комбинацию (доступен только при State Active)

    Чтобы задать задать карту нужно написать {"rank": "достоинство", "suit": "масть"}
    Достоинство и масть нужно писать с большой буквы по-английски
